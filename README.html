El Semicírculo Deslizante El contenedor `.item-icon` tiene `border-radius: 50%` y `overflow: hidden`. * `::before` pinta la mitad superior con un color pastel. Es estático. * `::after` pinta la mitad inferior con el color de acento, pero tiene aplicado un `transform: translateY(100%)`, lo que lo empuja fuera del contenedor (ocultándolo). * Al aplicar `li:hover .item-icon::after`, el transform cambia a `translateY(0)`, deslizándolo hacia arriba y completando el círculo con una transición de `0.4s`. ### 2. La Línea Ondulada Infinita Se utiliza un vector SVG muy estrecho (200x8px) que dibuja una onda sinusoidal. Este SVG está codificado en formato **Base64** y se inyecta directamente en la propiedad `background-image` del `::after` del título. * Inicialmente, el `background-size` es `0% 100%` (la onda es invisible). * En `li:hover`, el tamaño cambia a `200px 6px`, revelando la onda con una transición suave. * Simultáneamente, una animación `@keyframes wave-flow` desplaza el `background-position` de `0` a `200px`, creando la ilusión de que la onda se mueve hacia la derecha sin parar. --- ## 🎨 Guía de Personalización El archivo CSS está preparado para ser modificado fácilmente.
